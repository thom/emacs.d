# thom's .emacs.d

## Installation

```
$ cd ~
$ git clone git://github.com/thom/emacs.d.git
$ mv emacs.d .emacs.d
$ cd .emacs.d
$ git submodule init
$ git submodule update
$ cd vendor/rinari
$ git submodule init
$ git submodule update
```
